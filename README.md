iOS Project
====================

个人iOS开发项目，有部分功能待完善。

如果需要本地测试，需要搭建数据接口，可以在目录api内访问到测试接口数据（PHP）。另外，需要修改Models内的接口地址声明。

APP内接入了淘宝用户登录，需要修改defines.h内的Taobao Key和密钥。
